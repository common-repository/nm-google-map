This plugin is useful to add Google Map based on Address or Latitude and Longitude.

== Description ==

This is a google map plugin which is used to add google map through Address or Latitude and Longitude. If you have address then you can use that you can also use Latitude and Longitude to add google map.

There are also option to manage Zoom level, Width and Height of google map. You can add Info Window content too in this plugin.

There are 4 marker. Which is used for Google map marker.

There is shortcode to use this plugin in Page or post. Shortcode is [nm_show_google_map]


== Installation ==

1. Upload 'wp-post-url' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the WP Post Url 
4. There will a form where you will add addrees and other infomation.
5. Click on "Submit" button.
6. use [nm_show_google_map] to show map.
